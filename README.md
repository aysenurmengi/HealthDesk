HealthDesk, sağlık hizmetlerinin dijital ortamda daha verimli yönetilmesini amaçlayan bir web tabanlı platformdur. 
Sistem; hasta kayıtları, randevu planlama, personel yönetimi ve tıbbi verilerin güvenli takibi gibi temel işlevleri tek bir panelde bir araya getirmeyi amaçlar.


⚙️ Proje halen geliştirme aşamasındadır. Tamamlandığında frontend, veritabanı ve API dokümantasyonu eklenecektir.
